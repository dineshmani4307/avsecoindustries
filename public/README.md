# Public Assets Folder

Place your images here:

1. **logo.png** (or logo.jpg, logo.svg) - Your AVS ECO INDUSTRIES logo (will be used in the navbar)
2. **plate.jpg** (or plate.png) - Your areca leaf plate image (will be displayed as static image)

## Image Requirements:

- **Logo**: 
  - Formats: PNG, JPG, or SVG
  - Recommended size: 200x50px or similar aspect ratio
  - PNG with transparent background preferred
  - Will automatically try different formats if one fails

- **Plate Image**: 
  - Formats: JPG or PNG
  - Recommended size: 800x1000px or similar
  - Should show the oval areca leaf plate clearly
  - Will be displayed as a static image (no rotation)

The images will be automatically loaded by the website. Make sure the filenames match exactly (case-sensitive).
